O que significa Atributo?


Em HTML, um atributo é uma característica de um elemento da página, como o tamanho da fonte ou a cor. Os atributos são usados para amplificar uma tag. Quando um navegador da Web interpreta uma tag HTML, ele também procura seus atributos para que possa exibir os elementos da página da Web corretamente.

Os atributos fornecem informações adicionais sobre os elementos HTML. Por exemplo, os links HTML são definidos por uma tag. O endereço do link é especificado no atributo "href". Note que os atributos HTML são sempre incluídos entre aspas.

Os atributos HTML têm algumas características chave:
*Os atributos fornecem mais informações sobre os elementos HTML
*Os atributos são sempre especificados na tag inicial
*Os atributos são apresentados em pares nome/valor, tais como: name=value
